Visit the Bergm website

The Bergm package for R (available on CRAN: https://CRAN.R-project.org/package=Bergm) provides a comprehensive framework for Bayesian parameter estimation and model selection for exponential random graph models using the approximate exchange algorithm. It can also supply graphical Bayesian goodness-of-fit procedures that address the issue of model adequacy.
